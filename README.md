# Shittim-Chest

디스코드 기반 블루아카이브 유틸 봇

## 기능

### 1. 당번 추첨 (`/샬레당번추첨`)
- 특정 학교의 학생을 무작위로 선택
- 학교별 당번자 결정
- 옵션: 학교 선택 (선택 사항)

### 2. 생일 알림
- 매일 자정(서울 시간)에 학생 생일 자동 체크
- 등록된 알람 채널로 생일 정보 자동 전송

### 3. 알람 채널 등록 (`/알람등록`)
- 생일 알림을 받을 디스코드 채널 지정
- 서버별로 독립적으로 관리

### 4. 학교별 학생 데이터
아비도스, 아리우스, 게헨나, 하이랜더, 백귀야행, 밀레니엄, 붉은겨울, 산해경, SRT, 트리니티, 발키리, 와일드헌트

## 기술 스택
- Node.js
- discord.js v14
- node-cron (작업 스케줄링)
- dotenv (환경 설정)
- Jest (테스팅)

## 설치 및 실행

```bash
npm install
npm start
npm test
```

## 명령어 배포

```bash
node deploy-commands.js
```
